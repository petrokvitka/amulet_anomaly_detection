anomaly detection

in electric motors using autoencoder

machine learning

AMULET uses an Autoencoder to learn what an electric motor sounds like normally. If there are any deviations from the normal sound, AMULET will report an anomaly.

user friendly

We made AMULET easy to use: chose a sound file from your computer and AMULET will provide you with information about the motor state with 0.1 seconds step!

early anomaly detection

Innovative technology and precise data mining enables early detection of anomalies, which saves not only time, but also immense amount of finances.
